Detailed Description

This interface controls an AgentSystem. It provides methods to start and stop a game, and to get the next GameState from the AgentSystem.

Parameters
<E>Name of the custom implementation of IEnvironmentState that the environment uses. by providing this type information you can eliminate the need to cast
<A>Name of the custom implementation of IActionDescription that the environment uses. by providing this type information you can eliminate the need to cast
