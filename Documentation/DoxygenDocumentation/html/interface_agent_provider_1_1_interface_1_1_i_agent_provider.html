void AgentProvider.Interface.IAgentProvider.loadAgentSystem ( String  pathToAgentSystem) throws TechnicalException, SettingException

Sets the current agentSystem to the given one. Make sure this method is calls before any others of this interface. All calls to other will be executed to the lastly loaded agentSystem.

NOTICE: If no system was chosen, calling other methods of this interface will result in undefined behaviour!

Parameters:
pathToAgentSystemthe file path to the agentsystem to load
Exceptions:
TechnicalExceptionif a severe, technical error occurs
SettingExceptionif the application settings aren't accessible or corrupt
